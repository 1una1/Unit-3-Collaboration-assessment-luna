1.If i am collaborating with a classmate the first thing that each teamate would need to do is git branch and make their branch and git checkout to go into their own branch. This step is crucial, because it helps to make sure that everyone is working on their own things, so no one edits or gets rid of another persons work.

2. Developers use branches, because when working with other people you need to be able to work on your own copies without disrupting another persons work.

3. Git is a command line, which is used to track codes, its also the version control centor. Git and Github work together because with these two we can look at entire code bases/history.
